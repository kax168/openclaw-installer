# OpenClaw 一键部署工具

60秒部署你的私人AI助手。一行命令，全平台支持。

## 快速开始

**Linux / macOS:**
```bash
curl -fsSL https://raw.githubusercontent.com/kax168/openclaw-installer/main/install.sh | bash
```

**Windows (PowerShell):**
```powershell
irm https://raw.githubusercontent.com/kax168/openclaw-installer/main/install.ps1 | iex
```

## 自动完成

1. 检测操作系统和架构
2. 安装 Node.js 22
3. 安装 OpenClaw
4. 交互式配置向导（选择AI提供商、输入API Key、配置通道）
5. 注册系统服务（可选）
